Learn how to navigate the ML Triage environment and optimize your experiment analysis.

01. MISSION_OVERVIEW

You are an ML Triage Specialist. Your goal is to analyze a set of experiments, identify the best configurations, and diagnose failures using the available action engine.

STEP_ONE

Select a task from the sidebar. Each task has different objectives: finding best runs, identifying overfitting, or suggesting next-gen parameters.

STEP_TWO

Use the 'Investigate' action to unlock detailed logs for specific experiments. Check the reward trace to see if your actions are optimal.

STEP_THREE

Finalize the episode by submitting the best Experiment ID. Your performance is graded based on accuracy and efficiency.

Action Dictionary

Command Description Reward Impact

DISCARD Removes an overfitting or failed run from the active pool. Contextual (+/-)
SUGGEST Proposes a new hyperparameter config based on existing patterns. High Potential (++)
SUMMARIZE Closes the session with your final answer. Final Score Only
